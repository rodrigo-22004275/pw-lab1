HTML5 e CSS

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

header ❌ Indica cabeçalho.
Estava a usar antes mas substituí por nav.
nav ✔ Indica menu de navegação com links para as páginas.
Uso para apresentar os links de navegação e o logotipo.
main ✔ Indica conteúdo principal e único, que não se repete.
Uso para apresentar o conteudo principal do website.
article ❌ Indica porção do documento que pode ser vista de forma independente.
Não uso.
section ✔ Especifica uma parte do conteúdo, usada em conjunto com etiquetas h1|h2|...
Utilizo para separar secções de conteudo na main.
aside ❌ Indica anotações, publicidade ou outros assuntos desconexos.
Não uso.
footer ✔ Indica rodapé da secção (com copyright).
Uso para apresentar os créditos do website no fundo da página.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style. Também é possível, em vez de colocar a formatação dentro do documento, criar um link para um arquivo CSS que contém os estilos.

universal ✔ O seletor universal (*) aplica o estilo a todos os elementos da página.
Utilizo para colocar a fonte e margem inicial.
tipo ✔ O seletor de tipo usa o nome do marcador a estilizar.
Uso em vários elementos.
classe ✔ O seletor de classe associa um nome a um estilo.
Uso para estilizar apenas certos elementos como os títulos.
id ✔ O atributo id permite identificar univocamente um elemento do HTML (muito útil no JavaScript).
Utilizo em alguns casos como o #galleryIframe.
atributo ❌ Seletores que se aplicam a atributos de elementos.
Não uso.
pseudo-classe ✔ Seletor associado ao estado de um elemento.
Utilizo por exemplo o :hover na galeria de imagens para mudar a sombra.
pseudo-elementos ❌ Regras que se aplicam a partes de elementos.
Não uso.
